A great new source for late breaking deals is the Web. As web announcements can be made quickly, easily and globally, hotel "hot dates" sites are sprouting in cyberspace. Here is where to find them: The Hot Dates-Hot Rates page (accessible through MPI�s web site www.mpiweb.org or directly at www.hotdateshotrates.com ). allows you to search by city as well as region. Once you enter your name, phone and email, you can search by a city, region, block, dates and rates. This site links to the extensive TravelWeb site ( www.travelweb.com ) which has detailed information about the properties listed. Site visitors can be notified about new deals via email if desired. Once a property is found, on-line RFPs can be made. An estimated 500 hotels list via this service. This site yielded by far the most responses compared to the others listed in sample tests. PCMA�s "Open Dates � Discount Rates" page ( www.pcma.org/dates/search.htm ) gets the award for the cleanest layout. You can be search by date, geographic region, room block, rate and type of property. Currently over 250 hotel subscribers are listing information. However, in running test inquiries, very limited responses resulted. Once this is worked out, this could be a very useful site. Event Source ( www.eventsource.com ) contains a listing of number of hotels offering deals � some with links to hotel chain hot date pages. This includes the Hilton�s Value Rate program at www.hilton.com directed toward the leisure traveler. Many other hotels, including Sheraton, have similar site development programs in progress for both group and leisure travel. Another source for discounted group rates is through Adams/Laux�s MeetingNet ( www.meetingsnet.com/net/dates/ ). It lists 589 locations worldwide and allows searches by month, city, state, U.S. or international.
